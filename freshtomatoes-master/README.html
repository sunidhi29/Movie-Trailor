FRESH TOMATOES MOVIE TRAILERS

FRESH TOMATOES MOVIE TRAILERS is a web application made using object oriented Python programming. It includes the server side code to store a list of my favorite movies, including box art imagery and movie trailer URL.



INSTALL

This application was designed using Python 2 So you must have it installed, in case you already don't have. The link to install Python 2.7.13 is given below

Python 2.7.13 INSTALL

DIRECTORY STRUCTURE

The Movie_Trailer_Site folder consists of 3 files :

  • media.py : It consists of the class Movie in which we define 2 functions, the '__init__' function and the 'show_trailer' function.The __init__ function is used to initialise pieces of info like title,storyline etc.The show_trailer function makes use of the open function defined inside webbrowser module to open the youtube trailer of the movie whose instance is used to call this method.

    • >
  • fresh_tomatoes.py : This file has a function 'open_movies_page'.This function takes in a list of movies as input,and as output it creates and opens an html page or a website that shows the movies that we gave it in the another file,entertainment.py

  • entertainment.py : In this file we import the above two files- fresh_tomatoes.py and media.py .We create our movie instances like toy_story,avatar etc At the end of this file,we call the function 'open_movies_page' defined inside the file fresh_tomatoes.py .In this function , we pass an array of movies(instances) that we created as arguments.When we run this program , we get our Movie trailer website.


USAGE

Run the file entertainment.py to open the website.To make efficient use of this code and to play the youtube trailers click on the respective posters of movies
